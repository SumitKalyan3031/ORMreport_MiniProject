## Project Summary
This project implements an Online Reputation Management (ORM) study for the brand **Johnnie Walker**. It collects and analyzes social and review data to measure sentiment, identify recurring issues, and provide actionable recommendations to improve brand perception.

---

## Features
- Data aggregation from multiple social platforms  
- Sentiment and trend analysis  
- Interactive Power BI dashboards  
- Actionable ORM recommendations  

---

## Technologies & Tools
- Python (pandas, nltk)  
- Power BI  
- Excel / CSV  
- Google Alerts, Trustpilot, Social Mention  

---

## How to Run
1. Clone the repo and navigate to the folder.  
2. Place datasets in the `data/` folder.  
3. Run preprocessing and analysis scripts (if applicable).  
4. Open the Power BI dashboard file to view reports.
